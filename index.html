Mystery Number Game!

From the hit studio that brought you Nothing! Comes a game you didn't ask for!

At the start of the game you will be given a random number.

Below you will find a set of images I found using one of the many toolbars I can't uninstall from my browser. Each image has a random number associated with it.

To win this game, you must match the your score to the random number assigned. You will lose if your score goes above the the random number.

Be warned! Each time the game starts, the game will change the value attributed to these pictures.
